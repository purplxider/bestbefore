[BestBefore] 앱이 왜 필요할까요?

점점 늘어가고 있는 1인, 소형가구분들!
냉장고 관리, 잘 하고 계신가요? 상한음식들로 채우고 있진 않으신가요?

이제 냉장고 관리는, 이 앱으로 시작하세요!

사진 촬영으로 유통기한 자동 저장



  • 여태까지 유통기한을
    일일이 입력하셨나요?


    이제는 사진촬영만 하면
    우리가 직접 입력할 필요없이
    유통기한이 자동 입력됩니다.

D-Day 계산



  • 추가된 음식들을
    한눈에 알아보기 쉽게!


    리스트에 추가된 음식들은
    남아있는 유통기한을 자동으로 계산하여
    순서를 정리합니다.

로컬 알림



  • 등록한 리스트를 잊지 않기 위해!


    원하는 시간, 요일에 알람을 등록하면
    냉장고속 상황을 잊고 있더라도,
    유통기한 임박식품과 버려야할 식품을 알려줍니다.
